# el-popover

`el-popover` is used internally.

https://yujinpan.github.io/el-popover

## Usage

```
npm install --save el-popover
```

```vue
<template>
  <button>
    test
    <ElPopover placement="right">popover</ElPopover>
  </button>
</template>

<script>
import ElPopover from 'el-popover';

export default {
  components: {
    ElPopover
  }
};
</script>
```
