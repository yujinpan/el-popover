el-popover is used internally.

# Usage

npm i --save el-popover

import ElementUI from 'element-ui';
import ElPopover from 'el-popover';

Vue.use(ElementUI);
// Override registered ElPopover.
Vue.use(ElPopover);

Click me to show code
<template>
  <button>
    test button
    <ElPopover trigger="hover">test popover</ElPopover>
  </button>
</template>

<script>
export default {
  data() {
    return {};
  },
};
</script>

# Why

Do not change the original structure.

<template>
  <button>test button 1</button>
  <button>
    test button 2
    <ElPopover>test popover</ElPopover>
  </button>
</template>

It will additionally generate multiple dom levels.

<template>
  <button>test button 1</button>
  <ElPopover>
    test popover
    <button slot="reference">test button 2</button>
  </ElPopover>
</template>

# Options

Forward ElPopover (opens new window) all props/events/attrs.
